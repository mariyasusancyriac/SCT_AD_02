# TaskFlow - To-Do List App

TaskFlow is a simple and clean To-Do List Android application built using Kotlin and Jetpack Compose. It allows users to manage tasks efficiently with features like adding, editing, and deleting tasks. All tasks are stored locally using Room database.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Persistent data storage using Room Database
- Clean and simple UI

## Tech Stack

- Kotlin
- Jetpack Compose
- Room Database
- MVVM Architecture

## Project Structure

- `data/local` → Room database (Entity, DAO, Database)
- `data/repository` → Repository layer
- `viewmodel` → ViewModel logic
- `ui/screen` → UI screens using Compose

## How to Run

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run the app on emulator/device
